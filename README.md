## College Student Enrollment Review System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/1de00ad30fb04b589229efb282ef53f6/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Homepage
3. Personal Center
4. Student Status Information Management
5. Enrollment Processing Management



**Administrator Functions:**
1. Login
2. Homepage
3. Personal Center
4. Student Management
5. Student Status Management
6. Enrollment Processing Management
